# BillForge

Offline billing app with:

- Company name and logo on invoices
- Printable / PDF-save invoice view
- Signature blank at the bottom
- Customers, products, and invoice ledger
- Local backup export/import

## Run

Open `index.html` in a browser.

If you want a local server:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Print / Download

Open an invoice and click `Download / Print`. In the browser print dialog, choose `Save as PDF` if you want a file.
